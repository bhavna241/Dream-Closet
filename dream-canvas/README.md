# DreamCloset

## Project info

DreamCloset – Track Your Dreams & Goals.

## Getting Started

1.  **Clone the repository**

    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Install dependencies**

    ```sh
    npm install
    ```

3.  **Start the development server**

    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
